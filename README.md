# Geometric Anthesis

手の動きでスピログラフ状の花を咲かせるインタラクティブ作品です。

- 開始時にカメラ映像の上にタイトル画面が表示されます。
- Opening ののち、色・種・土・水を選んで、あなただけの花を咲かせます。
- 画面上の小さな緑の丸が、指さき（またはマウス）のしるしです。
- 緑の丸をつまむように指を近づける（ピンチする）と、この世界での「クリック」になります。

## ざっくりした操作の流れ

1. **Opening**
   - カメラに自分を映して、緑の丸（ポインタ）を確認します。
   - クリック / ピンチで物語のつづきへ進みます。
2. **説明シーン**
   - これから行う色・種・土・水の選択について、物語風に説明が表示されます。
   - 右下の「次へ」で色選択画面へ。
3. **色選択 / 種選択 / 土選択 / 水選択**
   - それぞれ一つずつ選んで「次へ」を押すと、次のステップに進みます。
4. **grow / bloom シーン**
   - 選んだ要素にもとづいて、花がスピログラフとして描かれ、咲いていきます。

## ファイル構成メモ

- `index.html` … p5.js と MediaPipe Hands を読み込み、`sketch.js` を起動するエントリ。
- `sketch.js` … シーン遷移、描画ロジック、手のピンチ判定などメインロジック。
- `ui.js` … UI 関連の補助コード（レイアウトやボタンなど）。
